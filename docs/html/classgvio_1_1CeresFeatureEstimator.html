int gvio::CeresFeatureEstimator::setupProblem ( )

Setup the optimization problem

It performs the following 3 tasks:

  1. Calculates an initial estimate of the landmark position
  2. Setup camera poses such that the first camera state is the origin, since here we are performing a bundle adjustment of a feature track relative to the first camera pose.
  3. Setup residual blocks
Returns
0 for success, -1 for failure

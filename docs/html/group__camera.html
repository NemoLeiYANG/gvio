Enumeration Type Documentation

enum gvio::TriggerMode
strong

Trigger Modes

Enumerator
NOT_SET 

Not set

FREE_RUN 

Free run (whenever the camera is ready)

SOFTWARE_TRIGGER 

Software trigger

TRIGGER_HI_LO 

Hardware trigger (falling signal edge). Enables camera memory mode in post-trigger

TRIGGER_LO_HI 

Hardware trigger (rising signal edge). Enables camera memory mode in post-trigger.

TRIGGER_PRE_HI_LO 

Hardware trigger (falling signal edge). Enables the pre-trigger in the memory mode

TRIGGER_PRE_LO_HI 

Hardware trigger (rising signal edge). Enables the pre-trigger in the memory mode

TRIGGER_HI_LO_SYNC 

Freerun synchronization / hardware trigger (falling signal edge). The freerun synchronization mode is currently only supported by the sensors of the UI-146x/UI-546x series

TRIGGER_LO_HI_SYNC 

Freerun synchronization / hardware trigger (rising signal edge). The freerun synchronization mode is currently only supported by the sensors of the UI-146x/UI-546x series

INVALID 

Invalid capture mode

Detailed Description

Multi-State Constraint Kalman Filter

This class implements the MSCKF based on:

Mourikis, Anastasios I., and Stergios I. Roumeliotis. "A multi-state constraint Kalman filter for vision-aided inertial navigation." Robotics and automation, 2007 IEEE international conference on. IEEE, 2007. APA

A.I. Mourikis, S.I. Roumeliotis: "A Multi-state Kalman Filter for Vision-Aided Inertial Navigation," Technical Report, September 2006 [http://www.ee.ucr.edu/~mourikis/tech_reports/TR_MSCKF.pdf]
